# Iteration 0.0.1 — 合同管理系统初始版本

## 目标

实现多用户合同管理系统的基础功能：用户注册登录、用户管理、合同 CRUD、PDF/Word 附件上传下载。

## 实施概要

### 数据模型
- User (id, username, password_hash, role, is_active, created_at)
- Contract (id, title, description, counterparty, amount, status, user_id, timestamps)
- Attachment (id, filename, original_filename, file_path, file_size, mime_type, contract_id, uploaded_at)

### 路由
- 认证: /auth/login, /auth/register, /auth/logout
- 合同: /, /contracts/create, /contracts/<id>, /contracts/<id>/edit, /contracts/<id>/delete
- 附件: /contracts/<id>/attachments/upload, /attachments/<id>/download
- 管理: /admin/users, /admin/users/<id>/toggle, /admin/users/<id>/delete
- 系统: /healthz

### 关键设计决策
- bcrypt 替代默认 Werkzeug 哈希（用户要求）
- Jinja2 模板 + Bootstrap 5 CDN
- 附件限制 10MB，仅允许 pdf/doc/docx
- 静态资源版本令牌 ?v=0.0.1
- HTML 响应 Cache-Control: no-cache

### 预置账号
- admin / admin123（管理员）
- demo / demo123（普通用户）
